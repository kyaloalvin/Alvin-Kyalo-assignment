# Alvin-Kyalo-assignment
> This project is a polished, portfolio-quality responsive photography landing page built during week two of the course. It intergrates semantic HTML structures alongside custom-made, clean CSS rules layout designs. The application presents an interactive design system displaying favorite media arrays, structured metadata lists, a project wireframe link, biographical details, and a working responsive contact form.
## Technologies used
* 'HTML'
* 'CSS'
## Project Setup instructions
To run this project locally on your machine:
1. Clone this repository to your computer system.
2. Open the Alvin-Kyalo-Assignment directory folder
3. Open 'index.html' inside any standard internet browser
